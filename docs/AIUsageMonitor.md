## 설치

1. `.dmg` 파일을 다운로드한 뒤 더블클릭합니다.
2. **AI Usage Monitor** 아이콘을 **Applications(응용 프로그램)** 폴더로 드래그합니다.
3. Launchpad 또는 Applications에서 앱을 실행합니다.

### 처음 실행할 때

코드 서명이 없거나 개발자 계정으로 서명되지 않은 빌드는 macOS 보안 경고가 표시될 수 있습니다.

- **우클릭 → 열기**를 선택하거나
- **시스템 설정 → 개인정보 보호 및 보안 → 확인 없이 열기**를 사용하세요.

- **메뉴바 커스텀** — 표시 요소와 상태별 글자 색상 변경, 실시간 미리보기, 최근 사용 색상 재적용
- Claude · Codex · Gemini는 로컬 기록에서, **Cursor는 서버 원시 이벤트와 로컬 기록**에서 실제 일일 토큰 사용량 수집
- **토큰 기록창** — 최근 12주 잔디, 월별 달력, 모델별 사용량, 주간 사용량 제공
- Claude · Cursor · Codex · Gemini별 **사용량·남은 사용량 위젯 8종** 지원
- AI별·종합 **토큰 잔디·월별 달력 위젯 10종** 지원
- **5분마다 자동 새로고침** — 수동 새로고침도 가능
- **Mac 부팅 시 자동 실행** (설정에서 켜기/끄기)
- **다크 모드** 지원
- Cursor / Codex / Gemini **앱 내 웹 로그인** — 로그인 완료 시 자동으로 창이 닫히고 데이터가 갱신됩니다
- **리셋 주기 프로그레스바** — 사용량 막대 아래에 리셋까지 남은 주기 또는 경과 정도를 함께 표시
- **리셋 페이스 색상** — 주간·Cursor 월간은 일 단위, 5시간 사용량은 분 단위 페이스로 색상 판정
- **사용량 구간 색상** — 주의·경고가 시작되는 사용량 기준을 직접 설정
- **사용량·잔여량 미리보기** — 선택한 관점에 맞춰 색상 구간값과 막대 방향을 확인

## 사용 방법

### 1. 앱 실행

앱을 켜면 **메뉴바 아이콘**과 함께 **메인 창**이 화면 중앙에 열립니다.  
메인 창을 닫아도 앱은 메뉴바에 계속 남아 있습니다.

### 2. 온보딩 튜토리얼

최초 설치 후 또는 새 버전으로 업데이트한 뒤 **튜토리얼**이 한 번 표시됩니다.

1. 상단 **2×2 그리드**에서 모니터링할 AI를 선택합니다.
2. 카드에서 **메뉴바에 표시할 사용량 항목**을 고릅니다.

튜토리얼을 설정에서 다시 확인할 수 있는 별도 메뉴는 없지만, 이후 카드와 설정에서 동일하게 조정할 수 있습니다.

### 3. 메뉴바

| **좌클릭** | 메인 창 열기 / 닫기 |
| **우클릭** | 컨텍스트 메뉴 (새로고침, 메인 창 열기, 종료) |

메뉴바에 표시되는 AI와 형식은 **설정 → 메뉴바 디자인**에서 바꿀 수 있습니다.

### 4. 메인 창

- **상단 토글**: 표시할 AI를 켜고 끕니다.
- **토큰 기록**: 독립 토큰 기록창을 엽니다.
- **서비스 카드**: 각 AI의 사용량, 리셋 시각, 요금제 등을 확인합니다.
- **톱니 아이콘(카드)**: 표시 항목 선택, **잔여 사용량 표시**, 로그아웃
- **하단 톱니**: 독립 설정창 열기
- **하단 새로고침**: 즉시 사용량과 토큰 기록 다시 불러오기

진행바를 클릭하면 해당 항목을 **메뉴바에 표시할지** 선택할 수 있습니다 (항목이 2개 이상인 경우).

**잔여 사용량 표시**를 켜면 사용량·리셋 막대가 채워지는 대신 **남은 사용량이 줄어드는** 방식으로 표시됩니다 (예: 1% 사용 → `사용 가능 99%`). Cursor **모델별 사용량** 영역은 이 설정의 영향을 받지 않습니다.

### 5. 위젯

데스크톱 또는 알림 센터의 위젯 편집 화면에서 **AI Usage Monitor**를 검색해 추가할 수 있습니다.

- Claude · Cursor · Codex · Gemini별 **사용량 위젯**과 **남은 사용량 위젯** 8종 제공
- AI별·종합 **최근 12주 일일 토큰 잔디 위젯** 5종 제공
- AI별·종합 **월별 토큰 달력 위젯** 5종 제공
- 사용량·남은 사용량·토큰 잔디 위젯은 가로형 중간 크기로 제공
- 월별 토큰 달력 위젯은 대형 크기로 제공
- 메인 앱의 사용량·토큰 기록과 막대 색상 설정을 위젯에 자동 반영
- 위젯을 클릭하면 앱의 메인 창 열기

### 6. 토큰 기록

메인 창 상단의 **토큰 기록** 버튼을 누르면 독립 토큰 기록창이 열립니다.

- **AI별·종합 보기**: 선택한 AI의 기록 또는 여러 AI의 합산 기록 확인
- **일일 토큰 잔디**: 최근 12주 동안의 날짜별 토큰 사용량 확인
- **월별 토큰 달력**: 선택한 달의 일별 사용량과 월간 총사용량 확인
- **모델별 토큰 사용**: 날짜별·월별 모델 사용량 확인
- **주간 토큰 사용**: 주 단위 토큰 합계와 수집 일수 확인
- **새로고침**: 각 AI의 기록을 다시 읽어 최신 토큰 사용량 반영

토큰 기록은 Claude Code, Codex CLI, Gemini CLI가 로컬에 남긴 실제 토큰 값과 Cursor 서버의 원시 사용 이벤트를 기준으로 집계합니다.  
대화 내용이나 프롬프트는 저장하지 않으며 날짜별 토큰 수와 모델별 합계만 앱 내부에 보관합니다.

Cursor는 서버에서 과거 일별 토큰을 불러오고, 이후 응답은 응답 완료 훅으로 실시간 기록합니다. 서버 기록을 우선 적용하며, 서버 조회가 실패할 때만 로컬 훅·구버전 DB 기록을 보조로 사용합니다. 최초 동기화와 토큰 기록창의 수동 새로고침은 이번 달 전체를 조회하고, 이후 자동 갱신은 최근 3일만 다시 확인합니다. Cursor 앱에 로그인되어 있어야 서버 기록을 불러올 수 있으며, 인증이 만료되면 토큰을 갱신한 뒤 조회를 재시도합니다. 앱 데이터를 삭제하면 설치된 훅과 토큰 로그도 함께 제거됩니다.

## 서비스 연동

### Claude (자동)

Claude Code CLI에 로그인되어 있으면 **별도 설정 없이** 자동 연동됩니다.

- macOS Keychain 또는 `~/.claude/.credentials.json`에 저장된 자격 증명 사용
- Claude Code 로컬 세션에서 실제 토큰 기록 가져오기
- 최초 조회 시 Keychain 접근 허용 창이 표시되면 **항상 허용**을 선택하세요.
- 로그인이 필요하면 터미널에서 `claude login`을 실행합니다.

### Cursor

**방법 A — Cursor 앱 (권장)**  
Cursor 앱에 로그인되어 있으면 로컬 세션으로 자동 연동됩니다.  
과거 일별 토큰 기록도 Cursor 서버에서 불러옵니다.

**방법 B — 웹 로그인**  
카드의 **Cursor 로그인** 버튼을 누르거나, 카드 하단에서 인증 방식을 **웹**으로 선택한 뒤 로그인합니다.  
세션이 감지되면 로그인 창이 자동으로 닫히고 사용량이 갱신됩니다.

Cursor 토큰 기록은 서버 원시 이벤트를 우선 사용하고, 이후 응답은 응답 완료 훅으로 실시간 수집합니다. 서버 조회가 실패할 때는 로컬 훅·구버전 DB 기록을 보조로 사용합니다.

### ChatGPT Codex

**방법 A — Codex CLI**  
터미널에서 `codex login`으로 로그인하면 로컬 연동됩니다.  
Codex CLI의 로컬 세션이 있으면 실제 토큰 기록도 함께 가져옵니다.

**방법 B — 웹 로그인**  
카드의 **Codex 로그인** 버튼으로 ChatGPT에 로그인합니다.  
Bearer 토큰이 감지되면 자동으로 창이 닫힙니다.

### Gemini

Gemini 사용량은 **웹 로그인**으로 연동됩니다.  
카드의 **Gemini 로그인** 버튼을 눌러 Google 계정으로 로그인하세요.  
한 번 로그인하면 이후에는 백그라운드에서 사용량을 불러옵니다.

Gemini CLI의 로컬 세션이 있으면 실제 토큰 기록도 함께 가져옵니다.

## 설정

메인 창 하단의 **톱니(⚙️)**를 눌러 독립 설정창을 엽니다.

| **일반 → 다크 모드** | 라이트 / 다크 테마 전환 |
| **일반 → Mac 부팅 시 자동 실행** | 로그인 시 앱 자동 실행 |
| **AI 표시 순서** | 드래그로 카드·메뉴바 순서 변경 |
| **메뉴바 디자인 → 표시 항목** | 아이콘·AI 이름·대괄호·항목 이름·퍼센트 기호를 각각 표시하거나 숨김 |
| **메뉴바 디자인 → 글자 색상** | 상태별 색상 변경, 실시간 미리보기, 최근 사용 색상 재적용·삭제, 기본 색상 초기화 |
| **사용량 막대 디자인 → 막대 표시 방식** | 사용량·잔여량 관점에 맞춰 구간값과 미리보기 방향 전환 |
| **사용량 막대 디자인 → 사용량 구간 색상** | 주의·경고가 시작되는 사용량 또는 잔여량 기준을 직접 설정 |
| **사용량 막대 디자인 → 리셋 페이스 색상** | 주간·Cursor 월간은 일 단위, 5시간 사용량은 분 단위 페이스로 색상 판정 |

각 AI 카드의 **톱니** 설정:

| **메인에 표시** | 카드에 표시할 사용량 항목 켜기/끄기 |
| **잔여 사용량 표시** | 사용량·리셋 막대를 잔여 사용량 기준으로 표시 (Cursor 모델별 사용량 제외) |
