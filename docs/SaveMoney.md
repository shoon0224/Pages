# 📱 아껴쓰자 (SaveMoney)

아껴쓰자는 **신용카드를 체크카드처럼** 쓰기 위한 iPhone 가계부 앱입니다.  
통장 잔액이 아니라 카드값·현금 소비·고정지출·목표 저축까지 반영한 **진짜 쓸 수 있는 돈**을 한눈에 확인할 수 있습니다.

![iOS](https://img.shields.io/badge/iOS-1.0.0-blue)

[![Download on the App Store](https://developer.apple.com/assets/elements/badges/download-on-the-app-store.svg)](https://apps.apple.com/app/id6783215620)

- 카드·현금 결제수단 구분, 카드값까지 반영한 잔고 표시

- 달력 — 일별 지출·입금 표시, 소비 강도 색상, 고정지출 포함/제외 토글
- 통계 — 카테고리별 상위 5 막대그래프, 입체 원그래프
- 내역 — 월별·필터(전체/지출/입금/카드/현금)·카테고리 필터
- 카테고리 관리 (지출·입금 카테고리 추가·수정·삭제)
- iCloud 자동 백업·동기화 (회원가입 없음, Apple ID 기준)
- iCloud 복원 대기 로딩 화면 (기존 데이터가 있을 때만 표시)
- 홈 화면 위젯 (2×2·가로형) — 오늘 쓸 수 있는 금액, 진짜 쓸 수 있는 돈, 남은 일수
- 잠금화면 위젯 (가로·원형) — 숫자 중심 표시, 탭 시 빠른 거래 기록
- 다크 / 라이트 / 시스템 테마 지원
- 온보딩 3단계 안내
- 데이터 초기화 (전체 / 입출력 내역만 / 커스텀 카테고리만)

## 💡 주요 기능

### 💰 진짜 쓸 수 있는 돈
- 통장 잔액이 아니라 **이번 사이클에 실제로 쓸 수 있는 금액**을 보여줍니다.
- 카드·현금 지출, 고정지출, 목표 저축을 모두 반영합니다.
- **남은 일수**와 **하루 사용 가능 금액**을 함께 계산합니다.
- 잔고 상태에 따라 안내 메시지·경고 색상이 바뀝니다.

### 📅 월 사이클 · 시작 보유 현금
- 매월 1일 또는 원하는 시작일(1~28일)로 예산 주기를 설정할 수 있습니다.
- 사이클이 바뀌면 **이번달 시작 보유 현금**을 입력해 새 주기를 시작합니다.
- 급여일·카드값 결제일에 맞춘 사이클 운영이 가능합니다.

### 🔁 고정지출
- 월세, 통신비, 구독료처럼 **매월 반복되는 지출**을 등록합니다.
- 결제일(1~31일, 말일)을 지정하고, 결제 완료 시 실제 거래로 기록합니다.
- 결제 완료된 고정지출은 **이중 차감되지 않도록** 계산됩니다.

### 📝 거래 기록
- **지출**(카드·현금)과 **입금**을 빠르게 기록합니다.
- 카테고리·메모·날짜를 함께 저장합니다.
- 홈 하단 **거래 기록** 버튼, 빠른 기록(카드·현금), 위젯 탭으로 즉시 입력할 수 있습니다.

### 📊 달력 · 통계 · 내역
- **달력** — 날짜별 지출·입금, 소비 강도 색상, 월간 요약
- **통계** — 카테고리별 지출 비중(막대·원그래프), 고정지출 포함/제외
- **내역** — 월별 목록, 유형·결제수단·카테고리 필터

### ☁️ iCloud 동기화
- 회원가입 없이 **Apple ID iCloud**에 자동 저장·동기화됩니다.
- 기기 변경·앱 재설치 후에도 같은 Apple ID로 데이터를 복원할 수 있습니다.
- 설정 화면에서 iCloud 동기화 상태를 확인할 수 있습니다.

### 📲 위젯
- **홈 화면** — 오늘 쓸 수 있는 금액, 진짜 쓸 수 있는 돈, 남은 일수, 상태 표시
- **잠금화면** — 가로·원형 위젯으로 핵심 숫자만 표시
- 위젯 탭 시 앱의 **빠른 거래 기록** 화면으로 이동

## 📖 사용 방법

### 처음 설정하기

1. App Store에서 **아껴쓰자**를 설치합니다.
2. 온보딩을 따라 앱의 핵심 개념을 확인합니다.
3. **설정** 탭에서 **이번달 시작 보유 현금**과 **목표 저축 금액**을 입력합니다.
4. (선택) **월 사이클** 시작일을 급여일·카드 결제일에 맞게 변경합니다.
5. **고정지출** 탭에서 월세·구독료 등 반복 지출을 등록합니다.

### 진짜 쓸 수 있는 돈 계산식

**진짜 쓸 수 있는 돈** = 이번달 시작 보유 현금 + 입금 − 지출 − 고정지출(미결제분) − 목표 저축

- 사이클 시작 **이전** 거래는 이미 시작 보유 현금에 반영된 것으로 봅니다.
- 고정지출 **결제 완료**로 기록한 거래는 지출에 포함되고, 같은 항목은 예약 차감에서 제외됩니다.

### 필요한 권한 및 시스템 설정

**iPhone**
- **iCloud** — 거래·고정지출·카테고리 데이터 동기화에 사용됩니다. **설정 → Apple ID → iCloud**에서 iCloud가 켜져 있어야 합니다.
- 카메라, 위치, 마이크, 연락처, 로컬 네트워크 등 **그 외 권한은 요청하지 않습니다.**

### 월 사이클 · 시작 보유 현금

**설정 → 월 사이클**
- **매월 1일** — 달력 월(1일~말일)과 동일
- **직접 선택** — 매월 1~28일 중 시작일 지정

시작일을 바꾸거나 새 사이클이 시작되면 **이번달 시작 보유 현금** 입력 시트가 표시됩니다.  
사이클 시작일 00:00에 로컬 알림이 올 수 있습니다.

### 고정지출 등록하기

**고정지출** 탭 → **+**
- 항목 이름, 금액, 결제일(1~31일 또는 **말일**), 결제수단(카드·현금) 입력
- 결제일이 되면 **결제 완료** 배지 표시
- 탭하여 결제 완료 기록, 또는 배지 옆 **수정**으로 날짜·금액 변경

### 거래 기록하기

| 방법 | 설명 |
|------|------|
| **홈 하단 「거래 기록」** | 지출 입력 시트 (기본 카드) |
| **빠른 기록 — 카드 / 현금** | 결제수단이 미리 선택된 지출 입력 |
| **위젯 탭** | 앱 실행 후 빠른 거래 기록 |
| **내역 · 달력 · 홈 최근 내역** | 기존 거래 탭하여 수정·삭제 |

입력 항목: 지출/입금, (지출 시) 카드·현금, 금액, 카테고리, 메모, 날짜

### 홈 화면

- **진짜 쓸 수 있는 돈** — 메인 잔고 카드, 스크롤 시 상단 고정
- **주의 카테고리** — 하루 사용 가능 금액 대비 과소비 카테고리 안내
- **요약 8칸** — 입금·지출·카드·현금·고정지출·목표 저축·가장 많은 지출 등 (셀 탭 시 설명 말풍선)
- **최근 입출금** — 최근 5건, **전체 보기** → 내역 탭

### 달력

- 월(또는 사이클) 이동, 날짜 선택 시 해당일 상세 내역
- **고정지출 포함** 토글 — 캘린더·요약에 고정지출 반영 여부
- 표시 모드 — 금액 / 소비 강도 색상

### 통계

- 월(또는 사이클)별 카테고리 지출 분석
- 상위 5 카테고리 막대그래프, 입체 원그래프
- **고정지출 포함** 토글

### 내역

- 월별 거래 목록, 월간 입금·지출 요약
- 필터: **전체 / 지출 / 입금 / 카드 / 현금**
- **카테고리 필터** — 지출·입금 카테고리별 조회
- 항목 탭 → 수정, 스와이프 → 삭제

### 카테고리 관리

**설정 → 카테고리 관리**
- 지출·입금 카테고리 추가, 이름·아이콘·색상 변경, 순서 조정, 삭제

### 위젯 추가하기

**홈 화면**
1. 홈 화면 길게 누르기 → **+** → **아껴쓰자** 검색
2. **2×2** 또는 **가로형** 위젯 추가
3. 탭하면 빠른 거래 기록으로 이동

**잠금화면**
1. 잠금화면 사용자화 → **위젯 추가**
2. **아껴쓰자** — 가로형 또는 원형 선택

### 화면 테마 · 기타 설정

- **설정 → 화면 → 화면 테마** — 시스템 / 라이트 / 다크
- **설정 → iCloud 자동 동기화** — 동기화 상태 확인
- **데이터 초기화** — 전체 / 입출력 내역만 / 커스텀 카테고리만 (확인 팝업 후 실행, **되돌릴 수 없음**)

## 🔒 개인정보 처리방침

**시행일:** 2026년 6월 24일

아껴쓰자 iOS 앱(이하 "앱") 이용과 관련하여 이용자의 개인정보를 어떻게 처리하는지 설명합니다.

### 1. 수집하는 개인정보

아껴쓰자는 **이용자의 개인정보를 개발자 서버에 수집·저장·전송하지 않습니다.**

- 회원가입, 로그인, 프로필 생성 기능이 없습니다.
- 이름, 이메일, 전화번호, 위치 정보 등 개인을 식별할 수 있는 정보를 개발자에게 전송하지 않습니다.
- 사용 통계, 분석, 광고, 추적 SDK를 사용하지 않습니다.

### 2. 기기 내·iCloud 저장

앱에서 입력한 거래·고정지출·카테고리·설정 데이터는 **이용자의 iPhone 및 iCloud 계정**에 저장됩니다.

- SwiftData + CloudKit을 통한 iCloud 동기화
- 위젯 표시용 요약 정보는 App Group(UserDefaults)에 저장
- 화면 테마 등 UI 설정은 기기 UserDefaults에 저장

이 데이터는 **이용자 본인의 Apple ID iCloud**에 속하며, 개발자 서버로 전송되지 않습니다.

### 3. iCloud 동기화

- iCloud 동기화는 Apple ID로 로그인한 기기 간 데이터 복원·동기화를 위해 사용됩니다.
- iCloud 사용 여부·용량·정책은 **Apple**의 iCloud 약관 및 설정을 따릅니다.
- iCloud를 끄면 해당 기기에만 데이터가 남을 수 있습니다.

### 4. 제3자 제공

아껴쓰자는 이용자의 정보를 **제3자에게 제공·공유·판매하지 않습니다.**

### 5. 보관 및 삭제

- 앱 내 **설정 → 데이터 초기화**로 기기(및 동기화된 iCloud 데이터)에서 삭제할 수 있습니다.
- 앱 삭제 시 기기 로컬 데이터는 제거되며, iCloud에 남은 데이터는 Apple iCloud 관리 방식을 따릅니다.

### 6. 아동의 개인정보

아껴쓰자는 만 13세 미만 아동을 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않습니다.

### 7. 처리방침 변경

본 개인정보 처리방침은 법령 또는 서비스 변경에 따라 수정될 수 있습니다.  
변경 시 이 페이지에 업데이트된 시행일을 함께 게시합니다.
