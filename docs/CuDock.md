# 📱 CuDock

CuDock은 iPhone을 Mac용 무선 매크로 패드·트랙패드로 활용할 수 있게 해주는 생산성 유틸리티 앱입니다.  
같은 Wi-Fi 네트워크에 연결된 Mac에서 앱 실행, 창 제어, 디스플레이 전환, 트랙패드 조작을 iPhone에서 수행할 수 있습니다.

## 💡 주요 기능

### 🎛️ 맥 매크로 패드
- Mac에 설치된 앱을 iPhone에서 바로 실행하거나 포커스할 수 있습니다.
- 실행 중인 앱을 한눈에 확인할 수 있습니다.
- 창이 여러 개 열려 있으면 개수가 표시되고, 창별로 관리할 수 있습니다.

### 🖱️ 원격 트랙패드 · 키보드
- 탭, 스크롤, 드래그 등 트랙패드 제스처로 Mac 커서를 제어할 수 있습니다.
- iPhone 키보드로 Mac에 글자를 입력할 수 있습니다.
- Mac에서 입력창을 클릭하면 iPhone 키보드가 자동으로 표시됩니다.
- 민감도를 조절할 수 있습니다.

### 🖥️ 디스플레이 · 창 제어
- 멀티 모니터 환경에서 디스플레이를 전환할 수 있습니다.
- 창 배치를 미리보고 원격으로 창을 닫을 수 있습니다.
- 창이 여러 개일 때 목록에서 원하는 창을 선택·이동·닫을 수 있습니다.

### 🔗 Mac 동반 앱
- iPhone 앱은 Mac용 CuDock 동반 앱과 함께 동작합니다.

## 📖 사용 방법

### 처음 설정하기

**Mac**
1. 위 링크에서 CuDock DMG를 내려받아 설치합니다.
2. 메뉴 막대에 CuDock 아이콘이 보이면 실행 중입니다. 종료하지 않아야 iPhone과 연결됩니다.
3. 메뉴 막대 아이콘 → **「연결」** 탭에서 손쉬운 사용 권한을 허용합니다.
4. **「앱」** 탭에서 iPhone에 표시할 앱을 검색해 추가합니다.
5. (선택) **「로그인 시 자동 실행」**을 켜 두면 Mac을 켤 때마다 CuDock이 자동 실행됩니다.

**iPhone**
1. App Store에서 CuDock을 설치합니다.
2. Mac과 같은 Wi-Fi에 연결합니다.
3. 우측 상단 **톱니(연결)** → 주변 Mac을 선택하거나 IP·포트로 직접 연결합니다.
4. 처음 Mac을 검색할 때 **로컬 네트워크** 권한 요청이 뜨면 **허용**을 눌러 주세요.

### 필요한 권한 및 시스템 설정

**iPhone**
- **로컬 네트워크** — 처음 Mac 자동 검색 시 요청됩니다. 같은 Wi-Fi의 Mac을 Bonjour로 찾는 데 사용합니다. 거부해도 Mac에 표시된 IP·포트로 직접 연결할 수 있습니다.
- 카메라, 위치, 마이크, 연락처 등 그 외 권한은 요청하지 않습니다.

**Mac**
- **손쉬운 사용** — 창 닫기(위 스와이프), 창 배치·자유 이동, 원격 트랙패드에 필요합니다. **시스템 설정 → 개인정보 보호 및 보안 → 손쉬운 사용**에서 CuDock을 허용하세요. **「연결」** 탭 **「창 제어 권한」**에서 상태 확인 및 **「권한 요청 / 다시 확인」**이 가능합니다.
- **방화벽** — 연결이 안 될 때 **시스템 설정 → 네트워크 → 방화벽**에서 CuDock 수신 연결을 허용하세요.
- **최초 실행(Gatekeeper)** — DMG 설치 후 차단되면 앱 우클릭 → **열기**로 실행합니다.

손쉬운 사용 없이도 **앱 실행·포커스, 앱 완전 종료, Mac 앱 등록·동기화**는 가능합니다.  
손쉬운 사용이 있어야 **창 닫기, 창 배치·이동, 원격 트랙패드, 창 목록·다중 창 관리**가 동작합니다.

### Mac에서 앱 등록하기

메뉴 막대 CuDock → **「앱」** 탭
- 검색창에 앱 이름 입력 → **+** 로 추가
- 상단 **1 · 2 · 3 …** 번호로 페이지 전환 (최대 5페이지)
- 빈 번호(점선) 탭으로 새 페이지 추가
- 페이지 2개 이상일 때 빨간 **−** 로 페이지 삭제 (확인 팝업)
- 앱 드래그로 순서 변경, 페이지 번호에 놓아 다른 페이지로 이동
- **×** 로 목록에서 제거
- **「단축 묶음 만들기」** — 앱 실행과 Mac 기능(미션 컨트롤, 화면 끄기, 소리 조절 등)을 하나의 아이콘으로 묶기

Mac에서 변경한 내용은 저장 버튼 없이 iPhone에 자동 반영됩니다.

### iPhone ↔ Mac 연결하기

**자동 연결 (권장)**
1. **톱니 → 연결**
2. **「주변 Mac (Wi‑Fi / 유선)」** 에서 Mac 선택
3. **「연결됨」** 이면 준비 완료

**직접 연결**
1. Mac **「연결」** 탭 **「수동 연결 정보」** 에서 IP·포트 확인
2. iPhone **「직접 연결」** 에 입력 후 연결
3. IP가 여러 개면 **「Wi‑Fi · 보통 이 주소」(192.168.x.x)** 를 먼저 시도

**연결이 안 될 때** — 같은 Wi-Fi 확인, Mac CuDock 실행 확인, 방화벽 허용, iPhone 로컬 네트워크 권한 확인, **↻** 새로고침 후 재시도

### 매크로 패드 — 앱 실행 · 포커스

- **탭** — 앱 실행. 이미 실행 중이면 Mac에서 해당 앱으로 전환
- **창이 2개 이상일 때 탭** — 창 목록 열기 (아래 「다중 창 관리」 참고)
- **좌우 스와이프** — 페이지 넘기기 (2페이지 이상일 때 하단 점 표시)
- **단축 묶음 탭** — 묶음에 등록된 앱·Mac 기능 순서대로 실행

**실행 상태 배지 (아이콘 왼쪽 위 점)**
- **초록** — 실행 중 + 창 열림
- **회색** — 실행 중 + 창 없음 (백그라운드)

**창 개수 배지 (아이콘 왼쪽 위 숫자)**
- 창이 **2개 이상** 열려 있을 때만 표시됩니다.

### 다중 창 관리

창이 2개 이상인 앱을 **탭**하거나, **위로 스와이프(창 닫기)** 하면 창 목록이 열립니다.

| 동작 | 설명 |
|------|------|
| **행 탭** | 해당 창을 Mac 앞쪽으로 가져오기 |
| **행 길게 누른 뒤 드래그** | 그 창만 모니터 미리보기로 이동·배치 |
| **행 오른쪽 ×** | 해당 창만 닫기 |
| **전체 창 닫기** | 열린 창을 한 번에 모두 닫기 |

창이 1개뿐이면 탭 시 바로 앱으로 전환하고, 위로 스와이프 시 앞쪽 창만 닫습니다.

### 실행 중인 앱 — 창 닫기 · 완전 종료

화면 하단 **「실행 중인 앱」** 스트립(그리드의 실행 중 아이콘도 동일)
- **탭** — 해당 앱으로 전환 (창이 2개 이상이면 창 목록)
- **좌우 스크롤** — 아이콘이 많을 때 (더 있으면 우측 페이드)
- **0.2초 길게 누른 뒤 드래그** — 창 배치·이동 (아래 참고)

**위 · 아래 스와이프**  
아이콘을 세로로 밀면 따라 움직입니다. 일정 거리까지 밀면 딱 걸리는 느낌(햅틱)과 **「창 닫기」** / **「완전 종료」** 라벨이 뜹니다. 그 상태에서 손을 떼면 실행됩니다.

| 방향 | 동작 | 조건 |
|------|------|------|
| ⬆️ 위 | 앞쪽 창만 닫기 (앱은 계속 실행) | **초록 배지**일 때만 |
| ⬇️ 아래 | 앱 완전 종료 (⌘Q) | 실행 중이면 초록·회색 모두 |

- **창 닫기(위)** — Safari 탭만 닫고 앱은 백그라운드 유지 (창이 여러 개면 창 목록)
- **완전 종료(아래)** — 앱 프로세스 자체 종료
- 반대 방향으로 되돌리거나 detent 전에 손을 떼면 취소

### 창 배치 · 디스플레이 이동

아이콘을 **0.2초 길게 누른 뒤** 드래그 → Mac 모니터 미리보기 표시 (손쉬운 사용 필요)

**위쪽 — 자유 이동**
- 위쪽 영역으로 끌면 창이 실시간으로 따라옵니다.
- 모니터 사이를 자유롭게 이동한 뒤 손을 떼면 그 위치에 놓입니다.

**아래쪽 — 창 배치**
- 모니터 미리보기 위에 놓으면 **전체 화면 / 좌측 50% / 우측 50%** 으로 스냅됩니다.

**단축 묶음** — 길게 눌러 드래그하면 묶음 패널이 열립니다. 안의 앱을 탭·스와이프·드래그할 수 있고, 바깥 탭으로 닫습니다.

### 원격 트랙패드 · 키보드

우측 상단 **트랙패드 아이콘** → 전체 화면 (손쉬운 사용 필요)

| 제스처 | Mac 동작 |
|--------|----------|
| 한 손가락 이동 | 커서 이동 |
| 한 번 탭 | 클릭 |
| 빠르게 두 번 탭 | 더블클릭 |
| 두 손가락 탭 | 보조 클릭 (우클릭) |
| 두 손가락 드래그 | 스크롤 |
| 하단 모서리 드래그 버튼 길게 누르고 이동 | 클릭한 채 드래그 |

**키보드 입력**
- Mac에서 입력창을 클릭하면 iPhone 키보드가 자동으로 표시됩니다.
- 우측 상단 **키보드 아이콘**으로 키보드를 직접 켜거나 끌 수 있습니다.
- 입력한 글자·Enter·Backspace가 Mac으로 전달됩니다.

- **민감도** 30% ~ 200%
- **「뒤로」** 로 매크로 패드 복귀

CuDock iOS 앱 및 Mac 동반 앱(이하 "앱") 이용과 관련하여 이용자의 개인정보를 어떻게 처리하는지 설명합니다.

### 1. 수집하는 개인정보

CuDock은 **이용자의 개인정보를 수집·저장·전송하지 않습니다.**

- 회원가입, 로그인, 프로필 생성 기능이 없습니다.
- 이름, 이메일, 전화번호, 위치 정보 등 개인을 식별할 수 있는 정보를 수집하지 않습니다.
- 사용 통계, 분석, 광고, 추적 SDK를 사용하지 않습니다.

### 2. 기기 간 통신

앱의 핵심 기능을 위해 iPhone과 Mac은 **같은 로컬 네트워크(Wi-Fi/LAN)** 안에서만 서로 통신합니다.

- Bonjour(mDNS)를 통한 기기 검색
- 로컬 IP 주소를 이용한 직접 연결(사용자가 입력한 경우)
- 앱 실행, 창 제어, 트랙패드 입력 등 기능 수행을 위한 명령 전송

이 통신은 **개발자 또는 제3자 서버로 전송되지 않으며**, 이용자가 연결한 iPhone과 Mac 사이에서만 이루어집니다.

### 2-1. iPhone 로컬 네트워크 권한

iPhone 앱은 같은 Wi‑Fi에 있는 Mac을 Bonjour로 검색하기 위해 **로컬 네트워크** 접근 권한을 요청할 수 있습니다. 이 권한은 기기 검색 및 연결에만 사용되며, 인터넷을 통해 개발자 서버로 전송되지 않습니다. 권한을 허용하지 않아도 Mac에 표시된 IP·포트로 직접 연결할 수 있습니다.

### 3. Mac 동반 앱 및 시스템 권한

Mac 동반 앱은 App Store 외 경로(DMG 등)로 배포될 수 있으며, iPhone 앱과 함께 동작하기 위해 Mac에서 **손쉬운 사용(Accessibility)** 등 시스템 권한이 필요할 수 있습니다.  
해당 권한은 Mac에서의 원격 제어 기능(창 닫기, 창 배치·이동, 트랙패드 입력 등) 수행에만 사용되며, 개발자에게 개인정보가 전송되지 않습니다.

### 4. 제3자 제공

CuDock은 이용자의 정보를 **제3자에게 제공·공유·판매하지 않습니다.**

### 5. 보관 및 삭제

수집하는 개인정보가 없으므로, 별도의 보관·삭제 절차가 필요하지 않습니다.  
앱 설정 및 연결 정보는 이용자의 기기에만 저장됩니다.

### 6. 아동의 개인정보

CuDock은 만 13세 미만 아동을 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않습니다.

### 7. 처리방침 변경

본 개인정보 처리방침은 법령 또는 서비스 변경에 따라 수정될 수 있습니다.  
변경 시 이 페이지에 업데이트된 시행일을 함께 게시합니다.
